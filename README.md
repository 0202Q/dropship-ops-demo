# Dropship Ops Demo (Prototype)

Clickable prototype showing how a messy dropship failure alert can be turned into a clean triage card with:
- Order + shop summary
- “Reason” labeling (unsellable vs address)
- One-click deep links (Shop Admin, Order Manager, AliExpress search)
- Refund request button (demo only)

## Live Demo
Once GitHub Pages is enabled, open:
https://o202q.github.io/dropship-ops-demo/

## What this is (and isn’t)
- ✅ Fake data, no Slack API, no CommentSold API
- ✅ Demonstrates the reduced workflow / fewer screens
- ❌ Does not place AliExpress orders
- ❌ Does not issue real refunds

## Run locally
Open `index.html` in your browser.
